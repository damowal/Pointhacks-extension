# Privacy Policy

**Last updated: February 2024**

## Overview

Point Hacks AutoFill is a browser extension that helps users fill credit card application forms. This policy explains how we handle your data.

## Data Collection

### What We Collect

The extension stores the following information that you provide:

- Personal details (name, date of birth, contact information)
- Address information
- Employment details
- Financial information (income, expenses, assets, liabilities)
- Identity document numbers (driver licence, passport, Medicare)

### Where Data is Stored

**Local Storage (Default)**
- All data is stored locally in your browser using Chrome's storage API
- Data never leaves your device unless you enable cloud sync

**Cloud Storage (Optional)**
- If you sign in with Google, your profile can be synced to Firebase
- Cloud data is associated with your Google account
- You can delete cloud data at any time by signing out

## Data Usage

Your data is used solely to:
- Auto-fill credit card application forms on supported bank websites
- Sync your profile across devices (if cloud sync is enabled)

## Data Sharing

We do not sell, trade, or share your personal information with third parties.

The following services may receive data when you use specific features:

| Service | Data Shared | Purpose |
|---------|-------------|---------|
| Addressify | Partial address text | Address autocomplete suggestions |
| Firebase | Profile data | Cloud sync (only if signed in) |
| Google | Authentication tokens | Sign-in (only if you choose to sign in) |

## Data Security

- All data transmission uses HTTPS encryption
- Firebase data is protected by Google's security infrastructure
- Local data is protected by Chrome's extension sandboxing

## Your Rights

You can:
- View all stored data via the extension popup
- Delete local data by clearing Chrome extension storage
- Delete cloud data by signing out of the extension
- Uninstall the extension to remove all local data

## Bank Website Interaction

When you use the "Fill Form" feature:
- The extension reads form fields on the current page
- Your stored profile data is written to matching form fields
- No data is sent to any server during this process

## Changes to This Policy

We may update this policy from time to time. Significant changes will be noted in the extension's changelog.

## Contact

For privacy concerns or questions, please open an issue on our GitHub repository.
